Never use abbreviations in memory files.

**Why:** Memory is read cold in future sessions without the context that made an abbreviation obvious. "RT" or "AC" or "SF" in isolation is ambiguous and degrades the usefulness of the memory. The user explicitly called this out as a hard rule.

**How to apply:** Write the full term every time — Record Type, Acceptance Criteria, Salesforce, User Story, etc. This applies to the body of memory files and to the one-line hooks in MEMORY.md.
